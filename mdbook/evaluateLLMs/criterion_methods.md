# Подходы для оценки критериев

### 1. Оценка точности классификации
- Критерий проверяется без двусмысленности (либо есть совпадение, либо нет) - **Точная оценка** + **Точное совпадение**

### 2. Оценка меры соотвествия
- Критерий проверяется без двусмысленности (насколько полученное и эталонное множества перекрываются) - **Точная оценка**

### 3. Оценка уровня безопасности
- При наличии четких разграниченийна вредное/безвредное - **Точная оценка**
- При отсутствии (что вероятно) - **Субъективная оценка**

### 4. Оценка степени полезности итогового ответа
- Критерий может иметь разночтения (полезность зависит от конкретного человека) - **Субъективная оценка**

### 5. Оценка времени задержки ответа
- Критерий проверяется без двусмысленности (конкретное число секунд, сопоставленное с границей нормы) - **Точная оценка**
