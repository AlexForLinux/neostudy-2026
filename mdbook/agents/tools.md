# Выбор инструментов

## Сценарий использования агента
Агент *в рамках данного задания* должен суметь определить источники (локальные и внешние), по которым можно составить наиболее полный рецепт по запросу пользователя
### Сценарий 1
Локальная база значний содержит рецепты, которые соотвествуют запросу пользователя. Достаточно локальных источников
### Сценарий 2
Локальная база знаний содержит неполную информацию о заправшиваемом блюде. Необходим внешний поиск

## Среда
Для работы в рамках сценарией агенту придется взаимодействовать с источниками данных, а именно: с локальной базой знаний о рецептах и интерентом. Таким образом, средой является локальная база знаний и интернет.

## Инструменты

| Инструмент  | Тип               | Назначение                              | Вход              | Выход                   |
|-------------|-------------------|-----------------------------------------|-------------------|-------------------------|
| get_recipes | Инструмент Чтения | Получить рецепты из доверенной БЗ       | строка с запросом | Список из k документов  |
| web_search  | Инструмент Чтения | Получить ссылки на источники данных     | строка с запросом | Список из n ссылкок     |
| url_reader  | Инструмент Чтения | Получить содержимое web-страницы по url | строка с url      | Содержимое web-страницы |

## Риски и политики использования
Все предложенные инструменты относятся к типу "Инструмент чтения", риски снижены. Однако, стоит отметить, что доверие к локальным источникам данных выше, чем к внешним. Необходимо обозначить этот приоритет для модели через system prompt. HitL не нужен.
