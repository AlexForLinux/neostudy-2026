# Формирование данных 

## Условия вызова
Необходим файл JSON формата с массивом запросов, расположенный по умолчанию в /tests/agent_is_working/queries.json
```
[
    {"query": "Как приготовить борщ?"},
    {"query": "Как приготовить винегрет?"},
    {"query": "Как приготовить чахохбили"}
]
```
## Вызов теста
```
py -m tests.agent_is_working.test
```

## Дополнительные параметры теста
```
py -m tests.agent_is_working.test 
    --inp <файл с запросами> 
    --out <вывод результатов> 
    --tries <число прогонов> 
    --sleep <пауза между запросами в сек> 
    --m <модель>
```
